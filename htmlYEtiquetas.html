¿Qué es HTML?

Este Lenguaje de marcado de hipertexto o HyperText Markup Language por sus siglas en inglés (HTML) se utiliza para estructurar nuestras páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que formarán parte de la página web como imágenes, listas, vídeos, links, etc.

¿Qué son las etiquetas?

Una de las características principales de HTML es que utiliza etiquetas para definir la estructura del contenido que se muestra en la página. Éstos son básicamente comandos que indican al navegador cómo mostrar el contenido en la pantalla.

Las partes principales del elemento son:

  1. La etiqueta de apertura: consiste en el nombre del elemento (en este caso, p), encerrado por paréntesis angulares de apertura y cierre. Establece dónde comienza o empieza a tener efecto el elemento —en este caso, dónde es el comienzo del párrafo—.
  2. La etiqueta de cierre: es igual que la etiqueta de apertura, excepto que incluye una barra de cierre (/) antes del nombre de la etiqueta. Establece dónde termina el elemento —en este caso dónde termina el párrafo—.
  3. El contenido: este es el contenido del elemento, que en este caso es sólo texto.
  4. El elemento: la etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al elemento.

Etiquetas básicas de HTML:

  • <head> para información sobre el documento.
  • <title> título en nuestro navegador.
  • <body> engloba nuestro código.
  • <H1>…<H6> para títulos dentro del contenido.
  • <p> para nuestros párrafos.
  • <strong> para poner el texto en negrita.
  • <br> para saltos de línea.
  • <div> división dentro del contenido.
  • <a> para enlaces.
  • <img> para añadir imágenes al documento.
  • <ol> para listas ordenadas, <ul> para listas desordenadas, <li> para elementos dentro de la lista.

<head> </head>

La parte superior del documento HTML, es donde podremos indicar los metadatos: título del documento, hojas de estilos, javaScript, CSS…

<title> </title>

Define el título del documento que se muestra en la pestaña de nuestro navegador. Sólo contiene texto.

<body> </body>

Indica la parte del cuerpo del contenido de un documento HTML. Es una etiqueta esencial para cualquier documento ya que engloba todo el contenido visible del documento.

<H1> </H1>...<H6> </H6>

Hay diferentes niveles de títulos, del 1 al 6. Nos permiten indicar la importancia del título y para estructurar el contenido, tomamos H1 como un título más importante y de mayor tamaño que un H6 de menor importancia y más pequeño.

<p> </p>

Anteriormente vimos esta etiqueta que es la más básica de HTML y sirve para nuestros párrafos.

<strong> </strong>

Si tienes mucho texto, es importante poder dar énfasis a una parte en concreto, con la etiqueta strong lo podemos hacer.

<br>

Con esta etiqueta HTML le podemos decir al navegador que viene un salto de línea. Nos sirve para hacer el texto más leíble. Aunque usar esta etiqueta no es recomendable, ya que nos puede traer problemas a la hora de darle formato a nuesto texto.

<div> </div>

Un elemento que es usado mayoritariamente para agrupar otros elementos y actuar como plantilla de otros controles. Esta etiqueta nos ayuda a estructurar el documento en secciones.

<a> </a>

Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde pondremos el enlace al que queremos conectar. Otro atributo muy usado es target, el cual nos sirve para indicar si el enlace se abrirá en una nueva ventana o en la misma.

<img> </img>

Usamos la etiqueta IMG para mostrar imágenes dentro del contenido. Necesita el atributo src para funcionar, ya que será donde indicaremos la ruta donde se encuentra guardada la imagen.

<OL> <li> </li> </OL> | <UL> <li> </li> </UL>

Las etiquetas OL y LI nos sirven para crear listas, OL para listas ordenadas y UL para listas sin orden. Dentro de las listas, los elementos se identifican con la etiqueta LI.
